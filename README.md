## Sobre o projeto

#####O desafio inicial era mostrar as gerações de games do pokémon e seus respesctivos detalhes.
#####O caminho seguido foi a implementação de uma arquitetura visual dinâmica que se adequa com a resposta da API pública https://pokeapi.co/ .
#####Com isso foi dada uma visão geral de uma consulta visual da API e por isso o nome do repositório ficou  biblioteca-pokemon .


## Instalação do projeto
```
yarn install
```

### Compilação para desenvolvimento 
```
yarn run serve
or
yarn start
```

### Compilação para produção
```
npm run build
```

### Rodando os testes
```
yarn test:unit
```
